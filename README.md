# 极简记账

一个简洁优雅的个人记账应用，使用纯HTML/CSS/JavaScript构建。

## 功能特点

- ✨ 极简设计，专注于核心记账功能
- 💾 本地存储，数据安全保存在浏览器
- 📊 实时统计本月收支情况
- 📱 响应式设计，完美适配手机端
- 🎯 快速记账，操作流畅

## 使用方法

直接在浏览器中打开 `plain_accounting.html` 即可使用。

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript（无框架依赖）
- LocalStorage API

## 部署

本项目已配置为可部署到 Vercel：

1. 将代码推送到 GitHub
2. 在 [Vercel](https://vercel.com) 上导入项目
3. 自动部署完成

## 本地开发

只需在浏览器中打开 `plain_accounting.html` 文件即可，无需构建步骤。

## 数据存储

所有记账数据都保存在浏览器的 LocalStorage 中，不会上传到服务器，确保隐私安全。
